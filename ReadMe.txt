项目名称：Unity 2.5D 综合能力演示 Demo
	这是一个基于 Unity 开发的 2.5D 动作/模拟原型，旨在展示在 解耦架构、内存优化、状态机控制 以及 数据持久化 方面的实践能力。

核心技术栈
	引擎版本： Unity 2022.3 LTS (URP)
	编程语言： C#
	核心功能：
	内存管理： 自主实现对象池（Object Pool），用于高效处理大规模怪物/物料生成，降低 GC 开销。
		AI 逻辑： 基于有限状态机（FSM）开发，实现具备待机、追踪、攻击逻辑的智能单元。
		导航系统： 利用 NavMesh 实现 2.5D 环境下的动态避障与路径规划。
		数据持久化： 集成 SQLite 本地数据库，实现生产日志/关卡分数的记录与查询。
		UI 框架： 采用 UGUI 搭建，实现响应式数据看板与动态弹窗。

项目亮点与重构历程
	解耦实践： 将原本耦合的逻辑重构为基于 事件中心（Event Center） 的广播机制，实现 UI 与战斗逻辑的彻底分离。
	配置文件驱动： 支持通过 JSON 外部配置动态调整怪物属性，具备基础的数值驱动开发意识。
